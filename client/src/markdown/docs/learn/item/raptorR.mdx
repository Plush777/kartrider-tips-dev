<Bq>랍토르 R이 이름에 걸맞게 스티어를 개조하여<br/>
오른쪽으로 방향을 조작할 때마다 스킬 게이지가 차오릅니다.<br/>
속도가 빠를수록 게이지가 더 빠르게 차오르며<br/>
게이지가 가득 차면 바리케이드를 획득합니다.<br/><br/>
OVERDRIVE 패치 이후로 랍토르 L의 스킬이 **빌런** 카트바디로 이식되었습니다.</Bq>

랍토르 L과 방향만 반대고 나머지 효과는 모두 동일합니다.

문제점 또한 동일합니다.

- 2025년 1월 2일 패치로 충전에 필요한 속력이 150km/h => 120km/h로 감소 되었습니다.
