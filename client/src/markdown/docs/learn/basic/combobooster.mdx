드리프트가 종료 된 시점에서 일정 시간 내에 전진 키를 누르면 순간 부스터를 사용할 수 있습니다.
<br/>

드리프트로 인한 감속을 어느정도 완화할 수 있지만,

부스터를 쓰며 주행 할 때 같이 사용하게 되면 오히려 감속이 더 심해질 수 있으니 부스터를 키지 않는 상황에서 사용하는 것이 좋습니다.
