<Bq>실수가 아닌 고의로 사고를 내는 기술.</Bq>

멀티를 하다보면 바짝 추격하던 상대 유저가 실수를 했는데, 

오히려 그 파편에 자신만 피해를 받고 정작 실수한 상대는 자신 덕에 사고가 회복되는 상황을

연구하여 테크닉화 시킨 것입니다.

<br/>

보통 1대1에서 도박수로 사용하거나, 팀전에서 미들 순위 디펜스용으로 사용하기도 합니다.

주로 사고가 났을 때 상대가 휘말리기 쉬운 좁은 지름길 구간이나,

유턴 구간이 고의 삽을 성공 시키기 쉬운 대표적인 구간입니다.
