<Bq>1위 레이서 앞에 바리케이드를 설치합니다. <br/>
바리케이드와 충돌 시 주행에 방해를 받으며, 충돌 후 사라집니다.</Bq>

사용 즉시 일자로 3개가 설치되며, 맵 지형에 따라 1개가 설치되는 경우 (지름길 혹은 좁은길) 도 있습니다.<br/>
1등이 코너를 돌 때 혹은 좁은 길에서 사용하는게 좋으며, 레일에 설치되는 경우 부딪혀도 감속이 되지 않기 때문에 <br/>
이러한 부분들을 고려하여 사용해야 합니다.

<br/>

팀전에서 같은 팀이 1등일경우에도 무조건 설치되기 때문에 이러한 상황에서는 바리케이드를 쓰지않거나, <br/>
보이스 채팅을 하고 있으면 팀원에게 바리케이드를 사용한다고 말해주는게 좋습니다.
