스킬이 준비되면 타키 머리 위에 별이 표시되며, 스킬로 들어가는 부스터 효과는 자신에게 적용되지는 않습니다.

반대로 스킬 발동 후 얻는 부스터는 주변에 아무도 해당 효과를 받지 않았을 경우 획득할 수 있습니다.

<br/>

버프 범위가 좁아서 사실상 부스터 획득이 메인이고 다른 레이서 부스터 부여는 덤이라고 봐도 되는데, <br/>
이를 역이용해 개인전에서 사용하는 유저들도 있는 편입니다.

<br/>

14초마다 부스터를 얻을 수 있어 러너로써의 성능은 좋아졌지만, 상대팀에 타냐 (스컬 캐논) 스킬이 있다면 조금 힘들 수도 있습니다.
