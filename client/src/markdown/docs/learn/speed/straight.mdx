<Bq>상대방의 라인을 공격하기 위해 부스터를 사용한 채 상대방을 향해 직진하는 기술.</Bq>

각종 멀티플레이 기술 중 가장 공격성을 띄는 기술입니다. 

상대보다 뒤쳐진 상황에서 코너 진입 시 OUT - IN 최적화 라인보다 

IN 라인 직진 부스터가 **코너 진입 자체는 빠르다**는 점을 이용하여 상대방과의 충돌을 노리고 인코스 직진 라인으로 들어가는 플레이입니다.

<br/>

테러 라인의 일종으로, 주로 아래와 같은 상황에서 많이 사용됩니다.

- 팀전에서 팀 순위가 안좋을 때 마지막에 순위를 뒤집어야 하는 경우
- 1대1 상황에서 불리한 경우
- 팀 순위가 유리한 상황에서 추격하는 상대방을 저지하고 아군과의 거리 차를 벌리기 위함
