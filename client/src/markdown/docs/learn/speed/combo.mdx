<Bq>드리프트 후 순간 부스터를 쓰는 동작을 계속 반복하는 기술.</Bq>

첫 부스터를 빠르게 모으기 위해 사용하는 기술입니다.

<br/>

영웅 등급 카트바디 업데이트 이후로 낮은 순간 부스터 가속으로 인하여 거의 안쓰는 기술이 되었습니다.

<br/>

현재는 빌리지 러시아워, 월드 뉴욕 대질주와 같은 출발선과 코너가 가까운 코스에서 첫 부스터를 모으는데 사용되기도 합니다.

#### 해당 기술이 특히 중요한 트랙들

A2 엔진 튜닝 순간부스터 강화 10 기준입니다.

- 빌리지 고가의 질주 / 붐힐 터널
- 사막 피라미드 탐험 / 빙글빙글 공사장 / 잊혀진 고대의 기록 / 오래된 송수관
- 공동묘지 어둠의 박쥐 성
- WKC 코리아 서킷 / 브라질 서킷 / 멕시코 투어링 랠리 / 일본 서킷 / 캐나다 서킷 / 상하이 서킷 / 싱가포르 서킷
- 카우지 추방자들의 마을 / 디스트릭트 13 / 오염된 황무지
- 해적 로비 절벽의 전투 / 상어섬의 비밀
- 월드 마이애미 드라이브 / 서울 남산 / LA 부스트우드 / 뉴욕 대질주 / 두바이 다운타운 / 신베이 투어 / 이스탄불 노을 광장
- 노르테유 전투 비행장
- 포레스트 지그재그
- 광산 골드러쉬
- 아이스 익스트림 경기장 / 360 타워
