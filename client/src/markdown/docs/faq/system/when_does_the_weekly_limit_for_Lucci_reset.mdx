UTC 기준 오전 12시 에 초기화됩니다.

- 최대 30,000 루찌를 획득할 수 있으며, 한계에 도달한 후 추가로 획득한 루찌는 50% 감소합니다.
- 레이싱 패스, 이벤트, 쿠폰과 같은 일반적이지 않은 루찌 보상은 주간 루찌 획득 한도에 포함되지 않습니다.

<Bq>PC방에서 플레이하면 주간 한도 없이 루찌를 무제한으로 획득할 수 있습니다.</Bq>
