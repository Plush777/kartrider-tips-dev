스킬 사용 시 브로디처럼 자신의 앞에 실드존을 설치합니다. 실드 존에 들어가게 되면 실드를 한번 부여받는데, <br/>
설명은 팀원을 서포트하는 역할인 것 같지만 개인 실드용으로 쓰는 경우가 많습니다.

<br/>

난이도는 초보자 추천으로 표기되어 있지만, 사용 시 실드를 바로 발동시키는게 아니라 브로디와 동일하게

실드를 받는 구역을 자신의 위치 바로 앞에 만드는 개념이라 공격 신호가 들어올 때 미리 사용해야 하며, 

코너 구간에서는 지형지물이나 방향에 의해 실드 발동이 취소되는 경우도 있기 때문에 상황도 따라줘야 합니다.

<br/>

숙련될 경우, 지피시장처럼 일정시간마다 공짜 실드를 사용하는 느낌으로 활용할 수 있습니다.

<br/>

스킬 구조 상 실드 효과를 부여 받는데 딜레이가 있기 때문에, 

물폭탄 및 사이렌과 같은 공격 아이템 혹은 미소, 네오, 토비와 같은 스킬 공격들을 즉각적으로 막을 수 없다는게 단점입니다.
