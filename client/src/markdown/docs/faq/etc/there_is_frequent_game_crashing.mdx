---
title: 게임이 자주 튕기는데 어떻게 해야하나요?
---

다음과 같은 경우 충돌이 발생할 수 있습니다.

- 네트워크 연결이 불안정합니다.
- 다른 프로그램과 충돌이 있습니다
- 해당 기기에 바이러스나 악성 코드가 있습니다.
- 해당 기기의 사양이 게임을 실행하기에 충분하지 않습니다. 시스템 요구 사항은 다음과 같습니다.

<br/>

<details>
    <summary>시스템 요구 사항</summary>

    - 운영체제: Windows 10 이상 (64bit)
    - 프로세서: 인텔 1세대 i3 / AMD FX-6000 시리즈
    - 메모리: 8GB RAM
    - 그래픽: GeForce GTX 760 / AMD Radeon HD 7950
    - 저장: 사용 가능한 공간 30GB
</details>

<br/><br/><br/>

- 충돌을 일시적으로 해결하는 방법

    - KartRider: Drift를 제외한 실행 중인 모든 프로그램을 닫습니다.
    - 이더넷 케이블을 사용하여 컴퓨터를 라우터에 직접 연결합니다.
    - 바이러스 백신 프로그램을 업데이트하거나 전체 검사를 수행하여 바이러스나 악성 코드를 확인하고 제거하세요.
