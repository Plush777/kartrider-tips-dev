키 세팅은 사용자의 취향에 따라 다릅니다. 

FPS 게임에 익숙한 유저들은 WASD를 선호하는 경우가 있고, 또는 넘버패드를 사용하는 경우도 있습니다.<br/><br/>
넘버패드를 사용하는 이유는 다음과 같습니다.

<br/>

1. 방향키로 할 때보다 어깨를 넓게 피고 할 수 있습니다.
2. 기계식 키보드의 경우 쓰는 숫자키를 제외하고 나머지 키들의 키캡을 뽑아서 컨트롤을 더 편하게 할 수 있습니다.
3. 옛날 (2005년 ~) 에는 넘버패드가 방향키보다 반응속도가 더 빠르다는 얘기가 있었습니다. (현재는 워낙 키보드들의 반응속도가 빨라져서 차이가 없습니다.) 

<br/><br/>

키 설정은 환경설정 - 조작법에서 원하는 키로 변경할 수 있습니다.
