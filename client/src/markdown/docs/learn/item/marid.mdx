<Bq>마리드가 부스터 아이템을 미리 구매해 레이스에 참여합니다. <br/>
부스터를 지닌 상태로 게임이 시작되며<br/>
일정 시간만큼 주행할 때마다 부스터를 획득합니다.<br/>
1등과의 거리 차가 커질수록 더 빠르게 획득합니다.</Bq>

확률적으로 부스터를 얻는 디지니와 달리 스킬 게이지만 채워지면 확정적으로 부스터를 얻을 수 있습니다.

100km/h 이상인 상태에서 주행만 하면 되기 때문에, 아이템전 실력이 부족한 초보자들이 자주 쓰는 편입니다.

<br/>

단점은 디지니와 마찬가지로 부스터를 수급하는 개념이라 아이템 슬롯을 자주 비워주어야 합니다.

슬롯이 꽉 차 있으면 부스터를 획득할 수 없습니다.
