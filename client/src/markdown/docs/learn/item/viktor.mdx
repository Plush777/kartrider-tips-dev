<Bq>투탑이 상대팀 레이서 중 한 명을 선택해 일정 시간 후에 폭발하는 폭탄을 부착합니다.<br/>
폭탄이 터지면 일정 범위 내에 있는 레이서들은 모두 폭발 피해 효과를 봅니다.<br/><br/>
<Link target="_blank" href="https://kartdrift.nexon.com/kartdrift/ko/news/update/view?threadId=2527539">2024/3/21 노르테유 패치</Link> 이후로 스킬이 추가되었습니다.<br/>
<Link target="_blank" href="https://kartdrift.nexon.com/kartdrift/ko/news/update/view?threadId=2574039">2024/5/30 패치</Link> 이후로 아군에게 폭탄을 설치할 수 없게 변경되었습니다.<br/>
OVERDRIVE 패치 이후로 투탑의 스킬이 **투 페이스** 카트바디로 이식되었습니다.</Bq>

지정된 레이서에게 시한폭탄을 심고 일정시간이 지나면 **커다란 폭발**과 함께 범위 내의 카트가 공중으로 뜨게 됩니다. 

풍선 장착 여부와 상관없이 무조건 떠오르기 때문에 적중시 상대방을 확실하게 묶어놓을 수 있으며, <br/>
폭연이 모두 사라질 때 까지 효과가 이어지기 때문에 폭발 후에도 근처에 지나가면 폭탄을 맞는 판정이 생기게 됩니다. 

범위는 대략 물폭탄과 비슷합니다.

<br/>

- 24년 5월 패치로 아군에게 폭탄을 설치할 수 없게 변경 되어 고의 팀킬은 불가능하지만, <br/>
상대한테 설치한 폭탄 폭발에 휘말리는건 여전히 피아 구분이 없으므로 주의해야 합니다.
