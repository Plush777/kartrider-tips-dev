마일리지의 주요 수급처는 **레이싱 박스** 이기 때문에, 레이싱 박스를 최대한 많이 오픈하여 아이템을 획득하는 것이 좋습니다.

<br/>

루찌는 충분한데 마일리지가 없는경우, <br/>
상점에서 제일 저렴한 올인원 박스 (개당 500루찌) 를 10개씩 구매하여 마일리지를 얻는게 효율적입니다.
