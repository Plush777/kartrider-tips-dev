<Bq>비비가 허공의 수증기를 불러 모아 눈덩이사태를 일으킵니다. <br/>
눈덩이들은 비비의 일정 거리 앞에 만들어지며,<br/>
맞게 될 경우 미끄러짐과 동시에 잠시동안 조작 불능 상태가 됩니다.<br/>
비비를 포함한 같은팀 레이서는 눈덩이에 피해를 입지 않습니다.<br/><br/>
<Link target="_blank" href="https://kartdrift.nexon.com/kartdrift/ko/news/update/view?threadId=2574039">2024/5/30 패치</Link> 이후로 아군에게 피격되지 않도록 변경되고, 쿨타임이 소폭 증가 하였습니다.</Bq>

눈덩이는 3개가 생성되며, 트랙의 가로 넓이를 전부 막을정도로 넓습니다.

눈덩이는 벽이나 장애물과 충돌하면 튕기며, 배찌의 롤링 물폭탄과 충돌해도 튕겨나갑니다.

<br/>

눈덩이가 생성되는 위치는 **물폭탄과 완전히 동일** 하며, 자신 위치 기준으로 오르막길에 설치 시 눈덩이가 내리막을 타면서 속도가 빨라져 <br/>
효율이 극대화되는 반면, 내리막길에서 쓰면 눈덩이가 느려지거나 멈춰버린다는 단점이 있습니다. <br/>

이러한 특성 덕분에 빌리지 시계탑이나 사막 피라미드 탐험과 같이 지름길을 못가면 손해가 큰 트랙에서 효율이 극대화 됩니다.

<br/>

2024/5/30 패치 이후로 아군에게 피격되지 않도록 변경되어 팀전에서도 팀킬 걱정없이 적극적인 활용이 가능해졌습니다. 

대신, 쿨타임이 소폭 증가하여 개인전에서의 활용도는 크게 떨어졌습니다.
