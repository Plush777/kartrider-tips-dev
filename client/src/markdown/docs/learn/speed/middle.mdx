date: 2024-06-25

<Bq>중위권에 머물며 상대를 디펜스 하거나 팀을 서포트하는 포지션.</Bq>

사고가 잦아 순위 변동이 심한 중위권에서 버티며 팀 포인트를 보조합니다.

중위권에서 머무르며 사고를 회피하고 상위권과 중위권의 격차를 벌려 놓는 것이 주된 목표입니다.

<br/>

스위퍼와 비슷하지만 팀원이 사고가 나서 뒤쳐진 경우, 미들이 대신 상위권으로 올라가기도 하기 때문에

스위퍼보다 지원의 성격이 더 강합니다.

<br/>

상위권으로 올라갈 수 있는 주행과 미들에서 버티는 몸싸움 둘 다 좋아야 합니다.
