아이템전에서는 튜닝 옵션 중 3가지의 옵션 (드리프트 가속, 드리프트 끌기 유지력, 출발 부스터 강화) 만 적용됩니다.
<br/>

아이템전에서 끌기는 큰 의미가 없기 때문에 드리프트 가속과 출발 부스터 수치를 최대로 올리는 것이 좋으며,

취향에 따라서 드리프트 끌기 유지력을 올리는 것도 좋습니다. 

<br/>

튜닝 옵션에 대한 설명은 <Link href="/docs/learn/tuning/speed">튜닝 - 스피드전</Link> 문서를 참고해주세요.
