대량 아이템으로 1등 역전을 차단하기에 좋습니다. <br/>
후방에서도 바리케이드 물량으로 어느정도 힘을 발휘할 수 있습니다. 

<br/>

그러나 바리케이드는 넓은 맵의 경우 효율이 크지 않아서 1등을 뺏기가 어렵고, 1등을 한다 해도 바나나 아이템이 잘 나오지 않기 때문에 <br/>
실질적으론 바리케이드 하나로 승부를 봐야합니다.

<br/>

이러한 점 때문에 실전성은 타 캐릭터에 비해 떨어지는 편입니다.

<br/>

+ 2024년 12월 5일, 아이템전 확정큐브 도입으로 바나나 혹은 바리케이드 큐브만 먹으면 이전에 비해
스킬을 많이 사용할 수 있게되었습니다.
