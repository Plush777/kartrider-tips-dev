카트라이더: 드리프트 도감 문서입니다. <br/>
도감 페이지에서는 모든 카트바디와 캐릭터 리스트를 확인할 수 있습니다.

<br/>

상단에 있는 사이드 바 혹은 아래 목차에서 원하는 항목을 선택하여 확인해보세요.

#### <Link href="/docs/encyclopedia/karts" type="component">카트바디 도감</Link>
모든 카트바디의 획득경로, 스텟 등을 확인해보세요.

#### <Link href="/docs/encyclopedia/characters" type="component">캐릭터 도감</Link>
모든 캐릭터의 획득경로, 코스튬, 스킬 등을 확인해보세요.
