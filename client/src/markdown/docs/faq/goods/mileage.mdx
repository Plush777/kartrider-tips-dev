플러스 박스를 포함한 모든 레이싱 박스에서 <br/>
중복 보유가 불가능한 유형의 아이템 (카트바디, 캐릭터 등) 을 중복 획득하는 경우, <br/>
아이템 대신 일정량의 마일리지를 획득하게 됩니다.

<br/>

중복 획득한 아이템의 등급이 높을수록 획득하는 마일리지의 양이 더 많아집니다.

### 마일리지 사용 방법

<Img className="staticImage" src="/images/ingame/img-mileage-use.png" alt="마일리지 사용 이미지"/>

상점 - 기타 탭에서 카트바디, 보너스권, 숙련도 플러스권 등의 아이템들을 구매하거나, <br/>
퍼스널 아울렛에서 사용할 수 있습니다.
