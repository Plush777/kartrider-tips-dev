<Bq>모비가 로두마니 해적단다운 재빠른 손놀림으로 상대의 아이템 슬롯을 베껴옵니다. <br/>
일정 시간마다 스킬이 활성화되며 부스터 효과가 주어집니다.<br/>
활성화 상태에서 아이템을 보유한 상대와 충돌할 경우<br/>
상대의 아이템을 복제해옵니다.<br/><br/>
단, 아이템 박스에서 획득할 수 있는 아이템만 복제가 가능합니다.</Bq>

일정시간마다 스킬이 발동되고, 동시에 10초동안 충돌한 상대의 아이템을 복제할 수 있는 상태가 됩니다. 

대신 다발 물폭탄이나 강화 사이렌 등 특수 아이템은 가져올 수 없습니다.

<br/>

부스터 효과와 템복제 효과는 사이렌의 부스터 / 공격 효과처럼 별개로 적용되어, 아이템 복제는 그대로 할 수 있습니다.

복제 상태는 10초간 지속되기 때문에, 앞에서 속도를 줄이거나 블로킹을 통해 아이템으르 복제할 수도 있습니다.

<br/>

팀전에서는 아군이랑 부딪혀도 효과가 발생합니다. 그래서 개인전과 달리 어떤 아이템을 복제할지 아는 상태에서 복사할 수 있습니다. <br/>
강탈이 아닌 **복제** 이기 때문에 천사나 번개같은 아이템을 가진 팀원과 부딪혀 해당 아이템을 복제하여 게임의 주도권을 잡거나, <br/>
승부를 뒤바꿀 수 있다는 장점이 있습니다.

<br/>

스킬 자체는 사기적이지만, 패시브 타입에다가 발동조건이 까다롭기 때문에 픽률 자체는 낮은 편입니다.
