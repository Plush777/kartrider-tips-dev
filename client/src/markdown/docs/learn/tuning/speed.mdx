스피드전에서는 모든 튜닝 옵션이 적용되기 때문에, 주행 스타일 혹은 포지션에 따라 다양하게 튜닝을 할 수 있습니다.



#### (아이템/스피드 공통) 캐릭터 특성 효과 활성화

<Division className="mdRatio">
    <Img src="/images/ingame/img-learn-costume.png"/>
</Division>

캐릭터 코스튬 사용 시 능력치를 추가적으로 얻을 수 있는 옵션입니다.

스피드 / 아이템 모두 적용되며, 각 코스튬마다 특성의 수치 및 종류가 다르며 높은 등급일수록 사용할 수 있는 특성이 많아지게 됩니다.

<br/><br/><br/><br/>

<Division className="mdRatio">
    <Img src="/images/ingame/img-learn-costume02.png"/>
</Division>

특성 효과를 사용하려면 다음 조건들을 모두 만족해야 합니다.

- 카트바디 튜닝에서 캐릭터 특성 효과 옵션을 활성화
- A2 엔진 카트바디 탑승

<br/><br/>

다음은 각 특성에 대한 설명입니다.

<br/>

- 기본 특성 : 코스튬에 상관없이 적용되는 특성으로, 조건에 상관없이 모두 적용됩니다.
- 아이템전 스킬 특성 : 아이템전 스킬에만 적용되는 특성입니다.
- 스피드전 주행 특성 : 스피드전에서만 적용되는 특성입니다.
- 보조 특성 : 부가적인 특성으로, 모드 상관없이 적용됩니다.

#### (스피드 전용) 부스터 자동 사용

<Division className="mdRatio">
  ㄴㄴ
</Division>

부스터를 연달아 사용 시 발생하는 **딜레이**를 없애는 옵션입니다. 
<br/>
해당 옵션은 활성화 하게되면, 첫 부스터 사용 후 다음 부스터 부터는 아이템 사용 키 (기본: ctrl) 를 꾹 누르고만 있어도 <br/>
부스터가 충분히 있을경우 끊기지 않고 계속 발동하게 됩니다.


#### 부스터 가속 / 드리프트 가속 / 부스터 시간 / 부스터 충전량

- 부스터 가속 : 부스터 사용 시 최고 속도까지 도달하는데에 걸리는 시간을 단축시켜줍니다.
- 드리프트 가속 : 끌기 드리프트 시 가속력을 높여줍니다.
- 부스터 시간 : 부스터 지속 시간이 증가합니다.
- 부스터 충전량 : 부스터 게이지 충전 속도를 상승시켜 부스터를 더 빠르게 채울 수 있습니다.

<br/>
<br/>

<Del className="mdDel">개인전에서는 부스터가 부족하기 때문에, 부스터 시간과 부스터 충전량을 6 이상으로 설정하는 것이 좋으며,<br/>
부스터 가속과 드리프트 가속은 웬만하면 최대로 설정하는 것이 좋습니다.
<br/>팀전의 경우는 부스터가 부족할 일은 잘 없기 때문에 부스터 시간 혹은 부스터 충전량의 수치를 줄이고,
다른 옵션에 투자하는 것이 좋습니다.</Del>

=> A2 엔진 업데이트 이후로 위 옵션들을 더 이상 튜닝할 수 없게 변경되었습니다.

#### 드리프트 끌기 유지력

<Division className="mdRatio">video</Division>

끌기 최고 속도가 높아지고, 쉽게 오래 유지할 수 있습니다. 
대신, 수치가 높을수록 드리프트를 끊기가 어려워지기 때문에 섬세한 조작이 필요합니다.

<br/>

드리프트에 대한 이해도가 높은 유저분들에게 추천드리는 옵션입니다. <br/>
밸런스가 있는 수치는 4에서 8 정도로 설정하는 것이 좋습니다.

<br/><br/><br/><br/>

#### 벽 충돌 시 부스터 게이지 보호

<Division className="mdRatio">video</Division>

드리프트 중 벽에 충돌 시 부스터 게이지의 일부를 보존합니다. <br/>
수치 1당 10% 이며, 최대치로 설정할 경우 벽에 충돌해도 부스터 게이지가 소모되지 않습니다.

<br/>

주행에 익숙치 않은분들에게 추천드리는 옵션입니다. 

본인이 벽에 잘 부딪힌다고 생각하시는 경우, 수치 5 이상을 추천드립니다.

<br/><br/><br/><br/>

#### 출발 부스터 강화

<Division className="mdRatio">video</Division>

출발 부스터 지속력 및 가속력이 증가합니다. <br/><br/>

주로 러너 성향 (주행에 자신이 있는) 유저분들에게 추천드리는 옵션입니다.
<br/>

초반을 빠르게 치고나갈 수 있다는 장점이 있지만, 출발 부스터는 일회용이기 때문에 이후에는 별 다른 효과가 없어서 <br/> 리스크가 어느정도 있는 옵션입니다.

<br/><br/><br/><br/>

#### 드래프트 가속

<Division className="mdRatio">video</Division>

드래프트 가속력이 증가합니다. <br/><br/>

디펜스를 좋아하는 플레이어 혹은 상대방을 빠르게 추월하고 싶은 경우에 추천드리는 옵션입니다.
<br/>

출발 부스터와 다르게 드래프트는 일회용 옵션이 아니므로 효과를 볼 수 있는 기회가 많으나, <br/> 해당 옵션 수치를 높게 설정했을 경우 출발 부스터 옵션에 적용할 수 있는 기어 수가 적어지기 때문에 스타트가 느릴 수 있습니다.
<br/>

이 옵션의 효과를 제대로 보려면 수치를 5 이상으로 설정하는 것이 좋으며, 밸런스를 원하시면 3 이상으로 설정하는 것이 좋습니다.
