앞 순위 상대를 잡는 것은 기존 물파리와 동일하지만, 물감옥이 아니기 때문에 버튼 연타로 빠르게 탈출이 불가능합니다. <br/>
물감옥에서 빨리 탈출할 수 있는 풍선이나 물감옥과 연관된 캐릭터들을 견제 하기가 좋으며, 공중에 뜰 수 밖에 없는 구간에서 발동시켜 <br/>
상대를 떨어뜨리거나, 점프 스킬을 가진 다오의 카운터로도 사용할 수 있습니다.

<br/>

대신 아이템 박스에서 물파리를 획득할 확률과 물파리가 책으로 변경되는 확률이 있기 때문에 어느정도 운이 필요합니다. <br/>

개인전보다는 물파리가 자주 나오는 팀전에서 더 효용성이 있습니다. 대신 물파리 수급을 위해 꾸준히 미들 순위를 타야하기 때문에 실력과 팀운이 중요합니다.
