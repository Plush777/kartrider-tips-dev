<Bq>자신과 같은 팀 모두에게 천사의 축복이 내려와 일정 시간 동안 방어 효과가 적용됩니다.<br/>
상대의 공격을 1회 방어하며 우주선과 자석은 방어할 수 없습니다.<br/><br/>
<Link href="https://kartdrift.nexon.com/kartdrift/ko/news/update/view?threadId=2147185" target="_blank">2023/6/8 패치</Link> 이후로 천사 아이템의 밸런스가 공격 아이템 1회 방어로 조정되었습니다.</Bq>

실드를 전체 팀원에게 적용시키는 아이템입니다. <br/>보이스 채팅이 아닌 상황에서는 보통 천사를 먹고 바로 쓰거나 
초반부에 나왔을 경우 눈치껏 팀원에게 공격이 가는거 같다 싶을 때 사용합니다.<br/>

보이스 채팅인 상황에서는 천사를 가지고 있다가 팀원이 "천!" 이라고 외치면 사용합니다.

### 관련 용어

- 천: 천사의 줄임말. 보이스 채팅에서 "천!" 이라고 외치면 천사 아이템을 써달라는 의미.
- 먹천: (보이스 채팅이 아닌 상황에서) 천사 아이템을 먹으면 바로 써달라는 의미.
- 상천: 상대팀 천사의 줄임말로, 상대 팀이 천사를 사용하였으니 잠깐 공격을 하지 말라는 의미.
