<br/>

단, 캐릭터 외형만 숨겨지는 것이고 무적은 아니기 때문에 물폭탄처럼 조준이 필요없는 공격은 회피할 수 없으며

벽 및 다른 레이서와 충돌하거나 스킬 사용 중 공격 아이템을 맞게 되면 스킬이 해제되니 주의해야 합니다.

<br/>

꼬마유령 잔상에 공격 아이템을 사용할 경우 효과없이 소멸되며, 상대팀에 미소, 투탑, 토비, 우니처럼 <br/>
조준형 공격 스킬을 가진 캐릭터가 2개 이상이라면 투명화 상태의 꼬마유령한테는 조준이 불가능 하므로 다오나 데릭에 비해 순위를 지키기 유리해집니다.

<br/>

스킬 효과 자체가 그렇게 크진 않기 때문에 개인전보단 팀전에서 팀원들의 디펜을 받으며 러너용으로 활용하기 좋습니다.
