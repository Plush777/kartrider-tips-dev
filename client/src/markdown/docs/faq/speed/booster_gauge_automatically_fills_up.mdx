---
title: 뒷순위로 가면 부스터 게이지가 자동으로 채워지는데 무슨 시스템인가요?
---

해당 기능은 <Link href="https://kartdrift.nexon.com/kartdrift/ko/news/update/view?threadId=2527539" target="blank">24/3/21 노르테유 패치</Link> 에서 추가된 견인 충전 시스템입니다.

- 1등과의 거리가 일정 이상 벌어졌을 때, 거리에 비례하여 부스터 게이지 자동 충전량이 증가합니다.
- 1등과의 거리가 더 크게 벌어질 경우, 부스터 사용 중일 때에도 부스터 게이지가 자동 충전됩니다.
