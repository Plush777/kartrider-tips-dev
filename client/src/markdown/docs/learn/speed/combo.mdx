첫 부스터를 빠르게 모으기 위해 사용하는 기술입니다.

<br/>

영웅 등급 카트바디 업데이트 이후로 낮은 순간 부스터 가속으로 인하여 거의 안쓰는 기술이 되었습니다.

<br/>

현재는 빌리지 러시아워, 월드 뉴욕 대질주와 같은 출발선과 코너가 가까운 코스에서 첫 부스터를 모으는데 사용되기도 합니다.
