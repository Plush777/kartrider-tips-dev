<Bq>토비가 친구인 스위퍼에게 도움을 요청합니다. <br/>

상대팀 레이서가 충돌할 경우, 강한 힘으로 튕겨냅니다.<br/><br/>
<Link target="_blank" href="https://kartdrift.nexon.com/kartdrift/ko/news/update/view?threadId=2574039">2024/5/30 패치</Link> 이후로 스위퍼가 적에게 충돌 시 공격하여 큰 넉백을 주도록 변경되었습니다.</Bq>

팩토리 맵에서 등장했던 스위퍼보다 훨씬 빠르게 움직이며, 플레이어와 한번 충돌하면 사라집니다. <br/>
길목이 좁은 곳일수록 벽과 벽 사이를 더 빠른 속도로 왕복하고 크기도 원본과 같을 정도로 크기 때문에, <br/>
앞쪽 길목이 좁은 곳에선 확정적으로 주행을 방해할 수 있으며 측면으로 충돌 시 플레이어를 멀리 밀쳐낼 수 있습니다.

<br/>

대신 방어아이템 (실드, 천사 등) 으로 방어가 가능하고, <br/>
팀전에서 아군이 충돌 했을 경우 넉백만 주지않고 바리케이드처럼 막히게 됩니다.
