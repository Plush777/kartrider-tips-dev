<Bq>배찌가 무엇이든 들어가는 만능 개구리 가방에서 <br/>
물풍선을 꺼내 자신의 앞으로 굴립니다. <br/>
레이서를 맞추면 물폭탄이 터지며, <br/>
반경 내에 있는 레이서는 물감옥에 갇혀 잠시 조작이 불가능해집니다. <br/>
물풍선을 오래 굴릴수록 피격당한 상대 레이서는 더 긴 시간 동안 물감옥에 갇힙니다.</Bq>

원작에서 플래그전 아이템이였던 <Link target="_blank" href="https://namu.wiki/w/%ED%81%AC%EB%A0%88%EC%9D%B4%EC%A7%80%EB%A0%88%EC%9D%B4%EC%8B%B1%20%EC%B9%B4%ED%8A%B8%EB%9D%BC%EC%9D%B4%EB%8D%94/%EC%95%84%EC%9D%B4%ED%85%9C#s-3.2.5">롤링 물폭탄</Link> 과 비슷하게 레이서와 부딪히지 않으면 최대 5초까지 굴러가다가 사라집니다.

<br/>

트랙 지형에 따라 물폭탄 경로가 매번 달라지기 때문에 표기된 난이도에 비해 훨씬 사용하기 어렵다는 평가를 받고 있습니다.

<br/>

팀 상관없이 모두 공격을 받을 수 있기 때문에 숙련자가 아니라면 팀전에서는 사용을 권장하지 않습니다.<br/>
이 때문에 팀전보다는 개인전에서 주로 많이 보이기도 합니다.

<br/>

숙련된 경우, 미리 방어 아이템을 쓰지 않는 이상 대처가 거의 불가능하기 때문에 순간적으로 순위를 빼앗기 좋습니다.
