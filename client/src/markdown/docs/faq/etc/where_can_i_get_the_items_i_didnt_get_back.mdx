<Img className="boxImage" src="/images/box/img-box.png" alt="박스 이미지"/>

얻지 못한 아이템들은 레이싱 박스에서 얻을 수 있습니다.

<br/>

단, 콜라보레이션 아이템 (BLACKPINK, BT21) 은 등장하지 않습니다.

### 상점

<Img className="staticImage" src="/images/ingame/img-where-can-i-get-the-items-i-didnt-get-back.png" alt="상점 이미지"/>

<br/>

- 올인원 박스 (카트바디, 캐릭터, 풍선, 번호판, 휠, 이모티콘, 이모션, 부스터, 스키드, 스티커가 등장)
- 나이스 올인원 박스 (카트바디, 코스튬, 풍선만 등장)
- 캐릭터 세트 박스 (코스튬, 이모션, 이모티콘만 등장)
- 카트바디 세트 박스 (카트바디, 휠, 부스터, 풍선, 스키드, 스티커만 등장)

### 싱글 미션

<Img className="staticImage" src="/images/ingame/img-where-can-i-get-the-items-i-didnt-get-back02.png" alt="싱글미션 보상 이미지"/>

<br/>

- 미션 올인원 박스 (해당 레이싱 박스에서 얻을 수 있는 카트바디, 캐릭터, 휠, 풍선, 번호판이 등장)

### 이벤트

<Img className="boxImage" src="/images/box/img-box-red.png" alt="박스 이미지"/>

<br/>

- 스패셜 코스튬 박스 1탄 / 2탄 (일부 코스튬)
- 스패셜 카트바디 박스 1탄 / 2탄 (일부 카트바디)

### 레이싱 패스

<br/>

- 프리미엄 플러스 박스 (이전 레이싱 패스 프리미엄 아이템)
