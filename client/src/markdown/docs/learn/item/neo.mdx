<Bq>네오가 광선검을 휘둘러 자신을 중심으로 원형의 파동을 만들어 냅니다. <br/>
파동에 맞은 상대 레이서는 공중으로 떠오르며 튕겨나갑니다.</Bq>

다른 캐릭터들에 비해 비교적 짧은 쿨타임과 생각보다 넓은 범위의 넉백 효과를 줄 수 있어 인기가 많은 캐릭터 중 하나입니다. <br/>
주로 상대방 앞에서 못가게 막는 플레이가 많은 편이고, 뒤에 있는 상대 위치를 어느정도 파악할 줄 알아야 합니다.  

<br/>

방사형으로 밀어내기 때문에 상대방보다 앞에있는 상태에서 맞추는게 제일 좋으며, 상대방보다 뒤에있을 때 사용하면 오히려 도와주는 꼴이 되니 주의 해야합니다.

대신, 속도 관련 능력이 없고 근거리 공격이기 때문에 앞서나가는 1등을 잡기 쉽지 않아서 개인전보단 팀전에서 미들 순위를 방해하는 역할을 맡게됩니다.<br/>
(네오의 스킬은 팀전에서 팀킬이 되지 않습니다.)
