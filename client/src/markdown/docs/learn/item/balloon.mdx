풍선을 착용하면 다음과 같은 효과를 받을 수 있습니다. 

풍선 등급이 높을수록 효과 개수가 많아집니다.

- (아이템전) 물감옥 빠른탈출 : 아이템전에서 탈출에 필요한 입력키 횟수가 감소합니다.
- (아이템전) 물감옥 빠른탈출 강화: 아이템전에서 탈출에 필요한 입력키 횟수가 더욱 많이 감소합니다.
- (아이템전) 벼락/우주선 피해 완화: 아이템전에서 벼락과 우주선 피격 시 피격 받는 시간이 감소합니다.
- (아이템전) 미사일 피해 방어 : 아이템전에서 미사일 공격을 미끄러짐 효과가 아닌 감속으로 방어합니다.
- (아이템전) 미사일 피해 완화 : 아이템전에서 미사일 공격을 미끄러짐 효과로 방어합니다.
- (스피드전) 부스터 증가 : 스피드전에서 자동 게이지 충전 속도가 증가합니다.
- (스피드전) 강화된 부스터 증가 : 스피드전에서 자동 게이지 충전 속도가 더욱 많이 증가합니다.
