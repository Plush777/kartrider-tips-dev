title: 차단 기능이 뭔가요?
---

<Img className="staticImage" src="/images/ingame/img-what-is-the-block-feature.png" alt="차단 이미지" />

플레이어의 특정 활동이나 커뮤니케이션을 차단합니다.

<br/>

※ 차단 목록 확인 및 해제는 메인화면 - 메뉴 - 차단 리스트에서 확인 및 해제가 가능합니다. <br/>
(차단 해제는 레이서명을 클릭하면 됩니다.)

- 같은 파티나 커스텀 레이스에서 플레이할 수 없습니다.
- 음성채팅이 차단됩니다. (양방향 차단)
- 친구로 추가할 수 없습니다.
- 멀티플레이에서 매칭이 불가능합니다.
