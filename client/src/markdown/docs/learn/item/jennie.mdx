<Bq>블랙핑크의 제니가 블링크에게 승리를 선물하기 위해<br/>
비장의 물폭탄을 준비해 레이스에 나섭니다.<br/>
제니는 다발 물폭탄을 지닌 상태로 게임을 시작하며,<br/>
물폭탄 획득 시 다발 물폭탄으로 변경됩니다.<br/>
다발 물폭탄 사용 시 물폭탄의 개수는 정해진 확률로 등장합니다.<br/><br/>
아이리와 스킬은 동일하지만, <br/>해당 캐릭터에는 <Link target="_blank" href="https://kartdrift.nexon.com/kartdrift/ko/news/update/view?threadId=2574039">2024/5/30 패치</Link>에서 아이리가 버프받은 능력 (일정 시간마다 물폭탄을 획득) 이 없습니다.</Bq>

아이리의 스킬과 동일합니다.
