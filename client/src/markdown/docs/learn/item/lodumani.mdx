<Bq>로두마니가 자신의 보물 1호인 악랄한 해골 지팡이를 휘둘러 <br/>
근방의 아이템 박스를 모두 약탈합니다. <br/>
약탈한 박스의 아이템은 현재 비어있는 슬롯의 갯수만큼 사용할 수 있습니다.</Bq>

쿨타임마다 거의 확정적으로 2개의 아이템을 획득할 수 있어, 스킬을 써서 아이템을 먹고 쫓아가거나 활용하여 <br/>
서포터형으로 쓸 수 있습니다.

다만, 아이템 슬롯이 최대 2개이고 효용성에 비해 너무 긴 쿨타임, 작은 발동범위 때문에 좋은 평가를 받고 있지는 않습니다. 

<br/>

아이템전을 하면서 템박스를 먹기 힘들다거나 투템 기술을 하기 어렵다면, 사용해보는 것도 나쁘지는 않을 것으로 보입니다.
