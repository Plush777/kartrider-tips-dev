<Bq>부스터 존을 순간 부스터와 함께 통과 후 앞 키를 뗐다가 다시 눌러 부스터를 재빠르게 사용하는 기술.</Bq>

부스터 존 통과 후 순간 부스터를 쓰면 더 빨리 가속을 얻을 수 있는데,

순간 부스터 이후 앞키를 바로 뗐다가 다시 누르면서 순간 부스터를 제거하고 재빠르게 부스터를 사용하여 가속을 얻을 수 있습니다.

<br/>

오리지널 카트라이더에는 없는 기술로, 

카트라이더 드리프트로 넘어오면서 순간 부스터 가속력이 낮아져 이를 보완하기 위해 해당 기술이 등장하였습니다.

천상계 타임어택에서 많이 쓰였으며, 
주로 부스터 존이 많은 월드 마이애미 드라이브나 노르테유 트랙들에서 사용됩니다.
