<Bq>오리온이 장애물에 정면으로 맞서며, <br/>
과거 러닝백으로 활약했던 면모를 가감없이 보여줍니다.<br/>
바리케이드 타입 아이템이나 스킬에 부딪히는 경우,<br/>
정의의 파동에 피격되는 경우 일정 횟수만큼 즉시 방어합니다.<br/><br/>
<Link target="_blank" href="https://kartdrift.nexon.com/kartdrift/ko/news/update/view?threadId=2527539">2024/3/21 노르테유 패치</Link> 이후로 방어 횟수가 1회 감소하는 대신, 네오의 스킬 공격을 추가로 방어할 수 있게 변경되었습니다.<br/>
OVERDRIVE 패치 이후로 오리온의 스킬이 **인더스틸, 프리저 A2** 카트바디로 이식되었습니다.</Bq>

바리케이드와 토비/네오의 스킬을 무시할 수 있게 해줍니다. <br/> 
바리케이드 같은 아이템들은 갑작스럽게 나오기 때문에 순간적으로 대처하기 힘든 초보자들에게 적합합니다. 

<br/>

바리케이드 5개를 설치하는 모스에게도 카운터이지만 그만큼 사용 횟수가 엄청 깎이기 때문에 주의해야 합니다.
