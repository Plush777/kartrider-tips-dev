<Bq>자신의 바로 앞 순위 상대 레이서에게 물파리를 날립니다. <br/>
물파리에 맞으면 물에 갇혀 잠시 조작이 불가능해집니다.</Bq>

물폭탄에 비해 갇히는 시간이 짧고, 물파리 역시 희귀 등급 이상의 풍선을 장착하고 있으면 <br/>
연타 횟수가 줄어들어 더 빠르게 탈출이 가능합니다.

<br/>

빌리지 시계탑 처럼 공중에서 떨어지면 빠지는 구간이나, 지름길 직전에 사용하여 <br/>
못타게 할 수도 있습니다. 언덕을 올라가고 있을 때 사용하는 것도 효과적입니다.
