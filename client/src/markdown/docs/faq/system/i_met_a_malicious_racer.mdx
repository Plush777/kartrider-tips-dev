title: 악의적인 레이서를 만났어요.

게임에서 악의적인 플레이어를 만난 경우 아래 방법을 사용하여 게임의 신고 기능을 통해 신고할 수 있습니다.

### 커스텀 게임

<Img className="staticImage" src="/images/ingame/img-i-met-a-malicious-racer-custom.png" alt="커스텀 게임 신고 이미지"/>

커스텀 게임 매칭을 준비하는 동안 신고하려면 신고하려는 사람을 클릭하고 신고 버튼을 클릭 하세요.

### 메인화면

1. 메인화면 메뉴에서 신고 버튼을 클릭하세요.
2. 가장 최근에 함께 플레이한 플레이어 목록이 표시되며, 신고하려는 사람을 클릭하면 됩니다.
3. 신고 사유를 선택하고, 신고 내용을 자세히 설명하여 신고하세요.
4. 신고 시 차단 버튼을 활성화하시면 해당 신고자가 차단됩니다.

### 소셜 메뉴

<Img className="staticImage" src="/images/ingame/i-met-a-malicious-racer-social.png" alt="소셜 신고 이미지"/>

소셜 메뉴에서 친구 추가 버튼을 클릭하세요.

#### 알아두세요!
- 악의적인 사용을 방지하기 위해 신고할 수 있는 횟수는 제한되어 있습니다.
- 1분에 10개가 넘는 신고는 허용되지 않습니다.
- 동일한 사용자에 대한 신고는 5개를 넘지 않아야 합니다.
