<Bq>미소가 레이서 중 한 명을 선택해 레전드 컵 에너지와 <br/>
미래 과학 기술이 결합된 능력을 실연합니다.<br/>
레이서가 우리팀일 경우 아무런 일도 발생하지 않지만<br/>
상대팀일 경우 미소에게로 끌려가게 되며, 미소에게 추가로 실드 효과가 적용됩니다.<br/><br/>
<Link target="_blank" href="https://kartdrift.nexon.com/kartdrift/ko/news/update/view?threadId=2527539#%EC%95%A1%ED%8B%B0%EB%B8%8C%EC%8A%A4%ED%82%AC">2024/3/21 노르테유 패치</Link> 이후로 스킨 등급 당 쿨타임 감소가 초 단위가 아닌 % 단위로 감소하도록 변경되었습니다.</Bq>

자석보다 지속시간이 짧지만 자력이 매우 강하여 직선 주로에서도 감속 효과가 번개 급으로 작용하며, <br/>
풍선 착용 상태로 미사일에 피격되거나 바나나를 밟는 등 조작 불능 상태에 빠졌다면 더 빠르게 뒤로 끌려가게 됩니다.

<br/>

자석과 다르게 거리가 조금 벌어진다 싶으면 효과가 끊기며 방어는 전자파가 아닌 실드 / 천사 등으로 방어할 수 있습니다. 

<br/>

너무 가까이에서 사용하게 되면 반작용 때문에 앞으로 튕겨나가 오히려 도와주는 꼴이 될 수 있으니 주의해야 합니다. <br/>
팀전의 경우에는 적에게만 효과가 있음에도 불구하고, 조준 자체는 아군에게도 들어가기 때문에 몰려있을 경우 신중히 조준해야 합니다. 

(아군에게 사용 시 아군에게는 아무런 영향이 없지만, 스킬은 사용한 것으로 간주되어 쿨타임이 발생합니다.)
