<Bq>전자파를 발사해 자신 또는 같은 팀의 자석 효과를 해제하고 우주선 공격을 방어합니다. <br/><br/>
<Link href="https://kartdrift.nexon.com/kartdrift/ko/news/update/view?threadId=2147185" target="_blank">2023/6/8 패치</Link> 이후로 전자파 아이템으로 자석 효과도 해제할 수 있게 변경되었습니다.</Bq>

아이템 슬롯이 부족하거나, 마지막 자석 싸움 맵 (팩토리 거내톱니 위협, 빌리지 운하처럼 직선 주로가 긴 트랙) 이 아닌 이상은 버리지말고 <br/>
팀원이 우주선에 공격받을 때 사용하는 것이 좋습니다. 

<br/>

단, 팀원이 사용한 자석은 해제할 수 없습니다.
