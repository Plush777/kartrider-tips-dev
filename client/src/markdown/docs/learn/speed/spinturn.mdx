<Bq>차체를 180도 돌리는 기술.</Bq>

일반적인 코너에서는 이런 기술을 쓰는 경우는 없고, 멋이나 묘기용으로 쓰거나 

멀티에서 몸싸움을 하다 카트바디가 뒤로 돌아갔을 때, 카트를 다시 앞으로 잡는 용도 (사고 회복용) 으로 쓰입니다.

(S자나 ㄹ자 코너, 광산 꼬불꼬불 다운힐 후반 구간과 같은 복합굴절 코너에서 쓰이기도 합니다.)

<br/>

아이템전에서는 <Link href="/docs/learn/item/attack/missile/#빽미" target="_blank">빽미</Link>로 뒷 주자를 공격하는 것으로 활용하기도 합니다.

### 조작법

1. 유턴하듯이 드리프트를 합니다.
2. 반쯤 돌면, 드리프트랑 앞 키는 떼고 뒷 키만 누릅니다.
3. 원래대로 돌리기 위해 뒷 키와 돌리려는 방향의 반대 키를 누릅니다.
4. 3번에서 드리프트 키만 눌러주면 뒤로 드리프트가 가능합니다.
5. 뒤로 드리프트 후 방향키로 방향을 잡아줍니다.

### 해당 기술이 많이 쓰이는 트랙들

- 카우지 추방자들의 마을
- 사막 빙글빙글 공사장
- 광산 꼬불꼬불 다운힐
- 빌리지 붐힐 터널
