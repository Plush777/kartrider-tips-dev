<Bq>루시드가 악몽의 나비떼를 소환하여 일정 시간 동안<br/>
속도 증가 및 자신과 충돌한 레이서를 공격하는 효과를 얻습니다.<br/>
스킬을 발동 중인 루시드와 충돌할 경우 부딪힌 레이서는 폭발 효과 피해를 봅니다.<br/><br/>
OVERDRIVE 패치 이후로 루시드의 스킬이 **밀키웨이** 카트바디로 이식되었습니다.</Bq>

사용 시 부스터 효과와 동시에 상대 레이서와 충돌 시 공중에 뜨는 효과를 부여합니다.

<br/>

자석 아이템과 연계하여 사용하면 큰 시너지를 발휘할 수 있습니다.

<br/>

디지니 / 마리드와 같은 부스터 수급형 캐릭터가 가지는 단점인

아이템 슬롯을 강제로 비워두어야 한다는 문제도 루시드의 경우는 자유로우며,

<br/>

일반 부스터를 획득하려면 추가 조건이 붙는 디지니 / 마리드와 달리 루시드의 경우는

지속시간이 긴 사이렌을 플레이어가 마음대로 사용할 수 있어서 상대적으로 긴 쿨타임을 제외하면

사실상 두 캐릭터의 상위호환이라고 볼 수 있습니다.

<br/>

가속형 스킬이기 때문에 공격용 말고 러너용으로도 활용이 가능합니다.

- 24년 12월 5일 패치로 카트바디에 스킬이 이식되며 기존 사이렌 효과 (풍선이 있을경우) => **무조건 대상을 공중에 띄우는 효과** 로 변경되었습니다. <br/>
이로 인해 스위핑 능력이 더욱 강화되었습니다.
