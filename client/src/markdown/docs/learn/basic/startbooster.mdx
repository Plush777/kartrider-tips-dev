<Division className="mdTip">
    <Paragraph className="mdTipText">카운트 후 Go가 나올 때 전진키를 바로 누르는게 아니라 <Strong>살짝 늦게 눌러보세요.</Strong> 
Go 글자에 파란색 이펙트가 생기면서 더 빠른 출발 부스터를 사용할 수 있습니다.</Paragraph>
