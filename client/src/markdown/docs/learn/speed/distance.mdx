<Bq>아군과 너무 붙어가지 않고 어느정도 거리를 유지하는 기술.</Bq>

팀전에서 아군과 상위권 (1위 ~ 3위) 에서 같이 달리는 경우, 아군과 너무 붙어가다가

누군가 실수를 하여 의도치 않게 팀킬이 되는 경우를 방지하기 위한 기술입니다.

<br/>

부스터를 살짝 끄거나 일부러 아웃 라인으로 가기, 드래프트 발동 시

부스터를 쓰지 않는다거나 등의 방법으로 거리를 유지합니다.
