<Bq>투투가 로두마니 해적단의 무서움을 보여주기 위해
비장의 무기를 준비하고 레이스에 나섭니다. <br/>
미사일 획득 시 일정 확률로 폭격 포탄으로 변경되며, <br/>
투투가 휘파람을 불면 전방에 폭격 범위가 생성되어 <br/>
범위에 들어선 누구든 미사일 공격 효과를 가진 포탄에 피격됩니다.</Bq>

잠시 동안 타격 영역이 바닥에 표시되고, 그 영역을 밟은 모든 레이서 (자신 포함) 에게 미사일 공격을 합니다.

미사일과 동일하게 풍선을 장착했을 경우, 피격 시 미끄러짐으로 대체됩니다.

<br/>

팀전의 경우는 팀킬 위험이 크기 때문에 안쓰는 편이 좋고, 개인전에서는 범위 공격으로 그나마 효과를 볼 수 있습니다.

<br/>

하지만 아이템 변경 확률이 높지도 않고 피격 시 무조건 공중에 떠오르는 투탑과 달리 풍선으로 피해를 줄일 수 있고,

자신도 피해를 받기 때문에 사용하기엔 애매한 부분이 있습니다.
