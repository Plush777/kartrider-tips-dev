<Bq>드라키가 상대팀의 액티브 스킬 게이지를 흡수해서 스킬 사용을 방해합니다.<br/>
흡수된 스킬 게이지는 드라키의 기본 속도를 향상시키는데 이용됩니다.<br/><br/>
<Link target="_blank" href="https://kartdrift.nexon.com/kartdrift/ko/news/update/view?threadId=2527539">2024/5/30 패치</Link> 이후로 스킬이 추가되었습니다.</Bq>

자신보다 앞서있는 레이서들의 액티브 스킬 쿨타임을 역행 (강제로 못쓰는 상태로 만듦) 시키면서 동시에 가속을 얻는 능력으로, <br/>
4인을 흡수했을 때 가속력이 상당하여 (부스터와 같이 사용할경우 최대 396km/h) 미들 순위 타기 좋고, 상황이 따라준다면 <br/>
선두권으로 순식간에 올라가거나 마지막 대역전도 가능합니다. 

<br/>

단, **액티브 스킬 게이지만 흡수** 하므로 패시브 스킬 캐릭터는 게이지가 있더라도 흡수할 수 없습니다. 

이러한 특성 때문에 상대편 캐릭터 조합에 따라 픽률이 달라지기도 합니다.

<br/>

스쿼드에서는 **쿨타임 역행** 이라는 효과가 진가를 발휘하는데, <br/>
드라키가 게이지를 흡수하는 시간 3초 + 쿨타임 역행으로 되돌리는 시간 3초를 합산하여 <br/> 
**6초간 액티브 스킬이 잠기는 효과** 를 상대팀에게 부여할 수 있습니다.

<br/>

즉, 상대팀의 스킬 전략을 망치는데 특화되어 있는 캐릭터라고 볼 수 있습니다.
