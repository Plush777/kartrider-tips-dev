---
title: 현재 1대장 카트바디가 뭔가요?
date: 2024-07-24
---

작성일 기준, 블랙핑크 콜라보레이션으로 출시되었던 **Pink Venom 몬스터** 카트바디입니다. 

<br/>

차체가 다른 카트바디들에 비해 크지만 민첩한 코너링과 <br/>
긴 부스터 지속시간, 빠른 탄력 드리프트, 탱크같은 몸싸움 능력들을 가지고 있어 여전히 인기가 많습니다.

<br/>

그 외에 아래 카트바디들도 고수들 사이에서 인기가 많습니다.

- 업사이드 다운 (작은 차체 / 민첩한 코너링 / 밸런스가 좋은 튜닝스탯)
- 멘티스 (민첩한 코너링, 높은 드리프트 가속)
- 리치 골드 (전체적으로 나쁘지 않은 스탯)
- 우주 탐사 (업사이드 다운보다 작은 차체. 하지만 부스터가 짧아서 팀전용으로 적합)
- 이클립스 알파 (높은 드리프트 가속)
- 파워포스 (높은 드리프트 가속과 긴 부스터 시간으로, 보급형 핑크베놈으로 불리고 있음)
- 워호그 (전체적으로 나쁘지 않은 스탯)

<br/>

카트바디에 대한 자세한 정보는 <Link href="/docs/encyclopedia/karts" type="component">도감 - 카트바디 도감</Link> 페이지를 참고해주세요.
