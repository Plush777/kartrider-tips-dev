<Bq>드리프트 시 각도를 최소한으로 줄여 적은 감속으로 드리프트를 하는 기술.<br/><br/>
숏 드리프트에 대한 설명은 <Link target="_blank" href="/docs/learn/speed/tech/short">이 문서</Link>를 참고해주세요.</Bq>

기본적으로 카트라이더는 물리엔진 특성 상 드리프트를 깊게 넣으면 그만큼 감속이 심해지고, 

얕게 넣으면 감속이 적어지는 특성을 가지고 있는데, 이러한 특성을 이용하여 나온 기술이 최적화 드리프트입니다.

<br/>

여기서 드리프트를 깊게 / 얕게 넣는다란, 드리프트의 각도는 드리프트 키와 방향키를 얼마나 오래 누르냐에 따라

결정되는데 깊게는 드리프트 키와 방향키를 오래 누르는 것, 얕게는 드리프트 키와 방향키를 짧게 누르는 것을 의미합니다.

<br/>

따라서 해당 기술에서는 드리프트 키를 **숏 드리프트가 나가지 않는 선** 에서 최대한 살짝 눌러주고,

반대 방향키를 눌러 드리프트를 끊어주면 됩니다.

#### 해당 기술이 많이 쓰이는 트랙들

- 빌리지 고가의 질주
- 사막 빙글빙글 공사장
- 월드 리오 다운힐
- 사막 오래된 송수관
