<Bq>직진을 하면서 부스터 게이지를 충전하거나, <br/> 드리프트 하기 전 적절한 각도로 차체 방향을 잡는 등을 위해 쓰이는 기술.</Bq>

한쪽으로 드리프트를 넣은 다음, 드리프트 키를 누르고 있는 채로 반대 방향키를 눌러 드리프트를 끊어서 사용합니다.

이 때, 먼저 넣은 방향키는 떼지 않아도 상관없으며 연속으로 쓰고자 한다면 드리프트 키는 계속 누르고 있어도 됩니다.

#### 해당 기술이 많이 쓰이는 트랙들

- 빌리지 고가의 질주
- 포레스트 아슬아슬 점프
- 아이스 설산 다운힐
- 공동묘지 어둠의 박쥐 성
- WKC 코리아 서킷 
- 월드 리우 다운힐
- 월드 시안 병마용
- 노르테유 전투 비행장
- 해적 상어섬의 비밀
- 카우지 폐기물 처리장
